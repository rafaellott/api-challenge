Como instalar
=============

Requisitos:

* virtualenv

* virtualenvwrapper

* Python 3.6


1 - Crie um ambiente virtual com o python 3.6

.. code-block:: bash

    $ mkvirtualenv forecast --python=/usr/local/bin/python3.6

2 - Instale os requisitos

.. code-block:: bash

    $ pip install -r requirements.
